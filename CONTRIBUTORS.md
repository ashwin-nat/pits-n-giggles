# Contributors

Pits n' Giggles is maintained by **Ashwin Natarajan** ([@ashwin-nat](https://github.com/ashwin-nat)).

This file exists because the git history does not tell the whole truth. Release PRs were
squash-merged into `main`, and squashing a multi-author pull request collapses every
contributor into a single commit authored by whoever performed the merge. The record below
is the correction.
